Hotel Reviews Prediction (LLM + ML)

TRAVEL & HOSPITALITY

Project Background

Hotel reviews are a critical voice of the customer in the hospitality industry directly influencing booking decisions and business reputation. With millions of reviews generated daily across Hotel platforms. Manual analysis of this feedback is impractical and time-consuming. Yet understanding nuanced guest sentiment beyond simple star ratings is essential for improving service quality identifying operational gaps and maintaining competitive advantage. This project addresses this challenge by developing an automated sentiment analysis system that combines LLM with traditional machine learning. Using DistilBERTa streamlined deep learning model we extract semantic meaning from raw reviews while handcrafted linguistic features (review length, punctuation patterns) capture stylistic nuances.

Problem Statement:

A 200-room hotel generates 500+ monthly reviews across platforms impossible to process manually. Existing tools misclassify 40% of mixed sentiments. Star ratings don't reveal why guests are dissatisfied and leaving hotels unable to prioritize fixes. This leads to Unaddressed service failures causing 34% revenue loss from repeat guests and inefficient resource allocation.

Objective:

  • To build a scalable sentiment prediction system that can help hotels analyze customer feedback automatically.
  • Reduce review analysis time from hours/day to minutes/day.

Contribution:

  • Performed extensive evaluation across multiple models and scenarios to compare performance.
  • Provided a web-based interface Streamlit app for demo and usability purposes

Scope:

  • This project uses a publicly available hotel reviews dataset containing text reviews labeled with sentiment classes (positive, neutral, negative) for training and testing the model.
  • The solution aims to help hotels automate feedback analysis, improve service quality and enhance the overall guest experience.
  • The model can be extended to other customer-facing industries like restaurants, airlines and e-commerce platforms making it adaptable for broader sentiment analysis applications.
