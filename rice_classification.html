Rice Image Classification

SUPPLY CHAIN

Project Background

Rice is an important food eaten all over the world. Sorting and classifying rice is very important in farming and supply chains. A proper rice classification system can help maintain quality, improve sorting and ensure consistent standards. This project uses a deep learning model called Convolutional Neural Network to classify images of rice into different types automatically. By using artificial intelligence and image recognition, this project reduces the need for manual work making classification faster and more reliable. This system helps farmers, suppliers and consumers by improving efficiency and reducing errors in sorting rice.

Problem Statement:

Currently rice is classified by people manually which takes a lot of time and can be inconsistent. Mistakes in sorting can cause problems in the supply chain such as poor quality control and incorrect stock management. Since there is an increasing need for automation and accuracy this project aims to create a reliable and efficient system to classify rice automatically.

Objective:

  • To create a machine learning model that can accurately classify different rice types.
  • Automate the rice sorting process to save time and improve efficiency.
  • Reduce human effort and mistakes in the classification process.
  • To make the model adaptable so it can classify other farm products in the future.

Contribution:

  • Automated Sorting: The model automatically classifies rice removing the need for manual sorting.
  • Higher Quality Standards: The model ensures rice is sorted consistently, improving quality control.

Scope:

  • This project uses the publicly available rice image dataset containing various rice types to train and test the model.
  • A custom CNN is designed to classify rice images into categories efficiently.
  • The solution is aimed at improving quality control and automating sorting processes in the supply chain.
  • The model can be extended to classify other agricultural products making it versatile for broader applications in the agricultural industry.

Steps Followed

  • Dataset Download: Downloaded the rice image dataset using kagglehub.
  • Dataset Inspection: Explored the directory structure to identify classes.
  • Image Visualization: Displayed random samples from each class.
  • Data Preparation: Created a DataFrame and split it into training, validation and test sets.
  • Data Preprocessing: Rescaled image pixel values and created data generators.
  • Model Design: Built a custom CNN model with convolutional, pooling, dense and dropout layers.
  • Model Compilation: Configured the optimizer, loss function and evaluation metrics.
  • Model Training: Trained the model with callbacks for optimization.
  • Model Evaluation: Assessed model performance using accuracy, loss and a classification report.
