About Me

Amey Pathare

– Data Analyst | Engineer

I help teams make decisions they can trust. My work is about taking messy, incomplete data and turning it into a clear picture that answers the questions whether that's a product change, a pricing shift or a process fix. I spend as much time listening to stakeholders and understanding their goals as I do cleaning and analyzing data because the best insights come from knowing which questions actually matter. I present findings in plain language with visuals that make the point obvious not just pretty. I'm comfortable moving between the technical details and the bigger picture and I measure my success by whether the people I work with feel more confident making a call after we've looked at the data together.

Python

Power BI

Build interactive reports that tell the story behind the numbers. Share dashboards with ease, guiding stakeholders towards data‑driven actions. Go beyond surface‑level insights with in‑depth filters and cross‑sections.

Excel

Transform messy spreadsheets into clean, organized structures for seamless analysis. Powerful data analysis accessible to everyone, regardless of resources. Design interactive dashboards for real‑time insights.

SSAS

Automate data refresh processes, freeing up time for deeper analysis. Connect SSAS with other BI tools and applications for a seamless data ecosystem. Leverage cloud‑based SSAS options for improved accessibility.

Oracle

Automate data refresh processes in Oracle using SQL scripts and scheduled jobs. Optimize Oracle queries and indexes to improve performance and efficiency. Connect Oracle with Tableau for seamless data visualization.

BigQuery

Utilize BigQuery’s serverless architecture for scalable analytics. Automate data ingestion and transformation using scheduled queries and Dataflow. Design optimized BigQuery datasets and tables for efficient querying.

Tableau

Build interactive dashboards that provide actionable insights. Connect Tableau with databases like Oracle, BigQuery and Snowflake. Leverage Tableau Server/Cloud for centralized data access and collaboration.

Snowflake

Utilize Snowflake’s cloud‑native scalability for high‑performance analytics. Automate data pipeline workflows using Snowflake Tasks and Streams. Optimize Snowflake queries with clustering and materialized views.

PySpark

Optimize Spark jobs with partitioning, caching, and efficient transformations. Integrate PySpark with cloud storage, data lakes and databases for large‑scale analytics.

R

Automate data cleaning and transformation using R scripts and tidyverse packages. Build statistical models and visualizations with ggplot2 and Shiny.

Databricks

Optimize Spark queries for large‑scale data processing and analytics. Connect Databricks with cloud storage and data lakes for unified data management.

Machine Learning

Automate data preprocessing and feature engineering using scikit‑learn, TensorFlow. Optimize model training with hyperparameter tuning (GridSearchCV, Optuna). Integrate ML models into production pipelines.

LLM

Automate text preprocessing and tokenization using Hugging Face Transformers, NLTK. Fine‑tune LLMs like BERT, XLM‑RoBERTa on domain‑specific datasets for improved performance.
